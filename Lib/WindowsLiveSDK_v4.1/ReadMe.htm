Windows Live SDK

Welcome to Microsoft Windows Live Software Development Kit (SDK). The Windows Live SDK contains OAuth WRAP Handler and Channel.htm designed to help you develop and build applications that connect to Windows Live.

The Windows Live SDK contains the following items:

  • OAuth WRAP Handler - Included in the SDK is OAuth WRAP Handler for ASP.NET Web Sites.
  • Channel.htm - Channel.htm file required for estabilshing cross domain communication in web applications
